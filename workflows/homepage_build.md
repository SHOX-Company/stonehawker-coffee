# Workflow: Homepage Build

## Objective
Build a high-converting, mobile-first homepage for Stonehawker Coffee that converts Instagram cold traffic into buyers.

## Section Order (Conversion Logic)
1. **Hero** — Attention + primary CTA ✅ DONE
2. **Proof Bar** — Instant trust signals ✅ DONE
3. **Products** — Desire + purchase intent
4. **Story** — Emotional buy-in / brand differentiation
5. **Roasts** — Deep product info / selection guide
6. **Reviews** — Social proof / overcome objections
7. **Offer** — Urgency / bundle / subscription upsell
8. **Footer CTA** — Final action + email capture

## Assets Needed
- `assets/images/hero.jpg` — Cinematic dark coffee hero (4:5 or 16:9, min 1400px wide)
- `assets/images/og-cover.jpg` — 1200x630 Open Graph share image
- Product photography (white/dark bg options)
- Brand mark / wordmark SVG
- Roast origin photography

## Files Touched
- `index.html`
- `css/main.css`
- `js/main.js`

## Mobile-First Checklist
- [ ] Hero CTA above the fold on 390px viewport
- [ ] Nav sticky with blur on scroll
- [ ] Buttons min 48px tap target
- [ ] Font sizes clamp for all breakpoints
- [ ] No horizontal overflow

## Performance Targets
- LCP < 2.5s (mobile 4G)
- CLS < 0.1
- No render-blocking resources
- Images: WebP, width/height set, lazy below fold
